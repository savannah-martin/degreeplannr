Required Classes

Computer Programming I

3 hours

CIS 171

Fall
Course description ›

A study of the design of algorithms, pseudocode, data types, control structures, arrays, program testing, and text file processing using a high-level programming language.

Corequisite: MAT 101 College Algebra.

Computer Programming II

3 hours

CIS 172

Spring
Course description ›

A continuation of CIS 171. Emphasis is on object-oriented constructs and techniques, generics and use of common data structures supported by the language.

Prerequisite: CIS 171 Computer Programming I.

Intro to Web Design

3 hours

CIS 211

Spring
Course description ›

An interdisciplinary introduction to web design covering both technical and artistic aspects. The course will cover the current tools and technologies used for web design and development. Emphasis will also be given to the artistic principles relevant to good web design. Students will design, develop, and critique websites.
